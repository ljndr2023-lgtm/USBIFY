USBIFY es una aplicación multimedia para Windows creada para reproducir música y vídeos MP4 de forma moderna y sencilla.

La aplicación permite escuchar canciones, reproducir vídeos en fullscreen, descargar contenido multimedia y organizar automáticamente tu biblioteca. No es obligatorio utilizar un USB para usar la aplicación. USBIFY funciona normalmente desde tu computadora, aunque también incluye opciones de sincronización con dispositivos USB si deseas copiar música o vídeos externamente.

La biblioteca principal muestra tus canciones y archivos multimedia organizados. Desde ahí puedes seleccionar contenido y reproducirlo fácilmente usando el botón PLAY o haciendo doble click sobre el archivo.

La sección “Videos” muestra únicamente archivos MP4. Al abrir un vídeo:

* se reproduce automáticamente en pantalla completa
* la aplicación recuerda el momento exacto donde dejaste el vídeo
* puedes continuar viéndolo después sin perder el progreso

Durante la reproducción fullscreen aparecen indicadores con:

* el tiempo actual y duración total del vídeo
* el volumen actual

USBIFY también incluye controles rápidos mediante teclado:

* ESPACIO → pausar o continuar vídeo
* Flecha izquierda → retroceder 10 segundos
* Flecha derecha → avanzar 10 segundos
* Flecha arriba → subir volumen
* Flecha abajo → bajar volumen
* ESC → salir del fullscreen

La aplicación soporta formatos multimedia como:

* MP3
* WAV
* FLAC
* MP4

USBIFY está desarrollado con Python, VLC, Tkinter y pygame para ofrecer reproducción multimedia rápida, moderna y estable.
