# USBIFY

Modern desktop music and MP4 video player built with Python.

## Features

- MP3 + MP4 support
- Fullscreen MP4 playback
- Resume video playback position
- 10-second seek controls
- Volume controls
- USB sync
- VLC powered video playback

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
py music_usb_VIDEO_CINEMA_PLUS.py
```
